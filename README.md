1. startup graveyard: a deep dive into the failures of startups 

2. Purpose: To analyze and decode what causes so many startups to fail each years and what are the reasons for which leads to their collapse, also deciding on what all factors are responsible for their failure 

3. Tech Stack
• 📊 Power BI Desktop – Main data visualization platform used for report creation.
• 📂 Power Query – Data transformation and cleaning layer for reshaping and preparing the data.
• 🧠 DAX (Data Analysis Expressions) – Used for calculated measures, dynamic visuals, and conditional logic.
• 📁 File Format – .pbix for development and .png for dashboard previews.

4. Data Source
source: kaggle datasets on failed startups
data on ~400 startups and their durations along with their fundings

5. Features / Highlights
The best dashboard explanation format. • Business problem • Goal of the dashboard • Walk through of key visuals (briefly!) • Business impact & Insights

 • Business Problem the startup scene is laced with so many dynamic factors which cannot be accounted for, this project aims to answer questions such as :
 which sector is the most affected by giants,competition and trend shifts or how much funding affects survival. which is very hard to answer through raw data 

• Goal of the Dashboard To deliver an interactive visual tool that: Enables users to explore various failed startups and takeaway what led to their ultimate demise through charts and visuals .
